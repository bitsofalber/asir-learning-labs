# ASIR Labs

Repositorio público con **laboratorios prácticos, apuntes y proyectos**
de la FP **Administración de Sistemas Informáticos en Red (ASIR)**.

## Contenido
- Sistemas Operativos
- Redes
- Bases de Datos
- Seguridad
- Automatización y scripts

## Objetivo
Compartir material práctico y documentado para estudiantes de ASIR
y personas que estén aprendiendo sistemas y redes.
